# Personal Portfolio - Jash Gohil

Welcome to my personal portfolio website repository! This portfolio showcases my work, skills, and experience.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
